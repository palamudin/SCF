# Parkour Animation Inventory

Source folder:

`Assets/SCF/Animation`

Preview tool:

`SCF > Setup > Add Parkour Animation Browser`

Enter Play Mode after setup. The on-screen panel shows one button per animation clip; each button caption is the clip name it plays.
